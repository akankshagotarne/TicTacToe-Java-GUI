# Tic Tac Toe - Java GUI

A simple Tic Tac Toe game built using Java Swing.

## Features
- Two-player mode
- Win & draw detection
- Score tracking
- Restart confirmation dialog
- Winning line highlight

## Technologies Used
- Java
- Swing
- OOP Concepts

## How to Run
Compile and run:

javac TicTacToe.java  
java TicTacToe
